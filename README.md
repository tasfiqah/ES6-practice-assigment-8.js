# JavaScript Fundamentals & ES6 Mastery

## 🚀 Overview
This repository serves as a technical demonstration of core **JavaScript (ES6+)** concepts. Developed as part of a MERN stack curriculum, this project showcases clean, efficient, and modern coding practices essential for scalable web development.

## 🛠 Tech Stack
- **Language:** JavaScript (ES6+)
- **Runtime:** Node.js
- **Key Concepts:** Array Methods, Object Destructuring, ES6 Operators, and Functional Programming.

## 💡 Technical Highlights
I have implemented the following logic to optimize data handling:
*   **Array Manipulation:** Utilized `push` and `pop` for dynamic data management.
*   **Functional Programming:** Leveraged `map()` and `filter()` for immutable data processing, ensuring cleaner and more predictable code.
*   **ES6+ Features:** 
    *   **Spread Operator:** For efficient array merging and cloning.
    *   **Rest Parameters:** Created flexible, scalable functions (e.g., dynamic sum calculation) that can handle any number of arguments.
*   **Object Property Handling:** Efficient usage of `Object.keys()` and property validation.
*   **Memory & Hoisting:** Demonstrated an understanding of JavaScript execution context and hoisting behaviors.

## 📂 Project Structure
- `assignment.js`: Contains all task implementations with documented outputs.
